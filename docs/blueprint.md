# **App Name**: Sina Estate Manager

## Core Features:

- Comprehensive Dashboard: Provides an overview of the estate's key data, including staff, residents, financial status, and more, with interactive charts and summaries.
- Interactive Estate Map: Allows administrators to view and manage villas on a graphical map, displaying villa details and resident information upon selection.  Supports adding, deleting, and rearranging buildings within the map interface.
- Personnel Management: Enables detailed recording and management of staff information, including contact details, personal data, documents, and financial records.
- Resident Directory: Maintains a searchable directory of residents, with fields for contact information, vehicle details, and villa assignments. Also tracks residency status.
- Financial Transaction Management: Tracks incoming and outgoing payments, manages payroll details, and generates detailed financial reports.
- Automated Payroll Calculation: Calculates payroll automatically, incorporating hourly rates, overtime, holiday pay, and deductions. It acts as a tool to reduce manual payroll processing.
- Document Management: Allows for the uploading, viewing, editing, and deletion of documents related to the estate, residents, staff, and external parties.

## Style Guidelines:

- Primary color: Deep Teal (#008080) for trustworthiness and stability, in accord with managing important records. DO NOT SUGGEST TEAL, unless the user explicitly requests it.
- Background color: Light beige (#F5F5DC) provides a soft, neutral backdrop that reduces visual strain.
- Accent color: Burnt orange (#CC6633) for calls to action and to draw attention to critical data points.
- Body and headline font: 'PT Sans', a modern humanist sans-serif for readability.
- Code font: 'Source Code Pro' for any code snippets related to the estate.
- Use a consistent set of professional icons for navigation and to represent various data types and actions.
- Implement a clear, intuitive layout with easy access to key features and data points.
- Use subtle animations and transitions to enhance user experience without distracting from core functionality.
